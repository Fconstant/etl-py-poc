BigQuery Glossary

Quick reference for BigQuery terminology. Updated as lessons progress.

Architecture

TermDefinition
OLAPOnline Analytical Processing. Queries scan millions/billions of rows to produce aggregations. BigQuery's workload.
OLTPOnline Transactional Processing. Fast reads/writes of individual records. Postgres's workload.
Columnar storageData stored column-by-column instead of row-by-row. Optimized for analytical queries that touch few columns across many rows.
CapacitorBigQuery's internal columnar storage format. Compressed, optimized for analytical scans.
DremelBigQuery's query execution engine. Turns SQL into a tree of parallel workers (slots).
ColossusGoogle's distributed storage system. Handles BigQuery's data storage, replication, recovery.
SlotA virtual CPU used by BigQuery's compute layer. More slots = faster queries.

Platform Features

TermDefinition
PartitioningSplitting a table into segments by a column value (usually date). Queries filtering on the partition column scan only relevant segments.
ClusteringSorting data within each partition by up to 4 columns. Queries filtering on clustered columns skip irrelevant blocks.
Materialized viewPre-computed query results stored on disk. Auto-refreshed by BigQuery. Faster reads, but limited SQL flexibility.
External tableReferences data outside BigQuery (GCS, Drive, etc.) without loading it. Query it directly with SQL.
Federated queryQuery external data sources (GCS, Cloud SQL, Sheets) using standard SQL, without importing into BigQuery.
EditionsCapacity pricing tiers: Standard, Enterprise, Enterprise Plus. Pay per slot-hour instead of per TiB scanned.
max_bytes_billedPer-query safety limit. Query fails if estimated scan exceeds this value. Prevents surprise bills.

Security

TermDefinition
IAMIdentity and Access Management. Controls who can do what on GCP resources. BigQuery has roles like dataViewer, dataEditor, jobUser.
Column-level securityHide specific columns from users who lack the required Fine-Grained Reader role.
Row-level securityFilter rows per user based on policy conditions. Different users see different subsets of the same table.

Data Model

TermPostgres EquivalentDefinition
ProjectDatabase serverTop-level GCP container. All BigQuery resources live under a project.
DatasetSchemaLogical container for tables. Controls access, location, and defaults.
TableTableStructured data in rows and columns. Has a schema.
ViewViewVirtual table defined by a SQL query. No stored data.
Materialized viewMateralized view (rare)Pre-computed results stored on disk. Periodically refreshed.
External tableForeign data wrapperReferences data outside BigQuery (GCS, Drive, etc.).

Pricing

TermDefinition
On-demand pricingPay per TiB of data scanned. First 1 TiB/month free. $6.25/TiB after that.
Capacity pricing (Editions)Pay per slot-hour. You reserve compute capacity. Cheaper at steady high usage.
Write dispositionWhat happens when loading data: WRITE_TRUNCATE (overwrite), WRITE_APPEND (add), WRITE_EMPTY (fail if exists).
Max bytes billedPer-query safety valve. Query fails if estimated scan exceeds this limit.

SQL

TermPostgres EquivalentDefinition
GoogleSQLANSI SQL (mostly)BigQuery's recommended SQL dialect. ANSI-compliant. Use this, not legacy SQL.
Legacy SQL—Older BigQuery SQL dialect. Deprecated. Avoid.
DMLSameData Manipulation Language: INSERT, UPDATE, DELETE, MERGE. Supported but not BigQuery's strength.
DDLSameData Definition Language: CREATE TABLE, DROP TABLE, ALTER TABLE.
QUALIFYSubquery wrapperFilters window function results inline. No Postgres equivalent — simplifies ROW_NUMBER/RANK filtering.
SELECT * EXCEPTColumn listSelect all columns except named ones. Useful after window functions that add temporary columns.
UNNESTunnest()Expands an ARRAY into rows. Used with cross joins: FROM t, UNNEST(arr) AS elem.
SAFE_DIVIDENULLIF(x,0) trickReturns NULL on division by zero instead of failing. Use proactively.
SAFE_CASTTRY_CASTReturns NULL on cast failure instead of failing.
CTE (WITH)Same syntaxCommon Table Expression. In BigQuery, inlined (re-scanned per reference), not materialized.
